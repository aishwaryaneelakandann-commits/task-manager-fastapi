A professional task manager CRUD app built with FastAPI (Python backend) and modern HTML/CSS/JavaScript (frontend).
## Setup
1. Clone this repository:
   git clone https://github.com/aishwaryaneelakandann-commits/task-manager-fastapi.git
2. Install Python dependencies:
   pip install -r requirements.txt
3. Run the backend:
   python main.py
4. Open TaskManager.html in your browser.
- main.py: FastAPI REST backend
- models.py, database.py, schemas.py: Database models and logic
- requirements.txt: Python dependencies
- TaskManager.html: Frontend UI
